# Telegram интернет-магазин бот

Бот для Telegram, который позволяет:
- Просматривать каталог товаров
- Добавлять в корзину и оформлять заказ
- Выбирать дату доставки с проверкой праздников
- Использовать кнопки “Купить 🛒”, “Подробнее ℹ️”, “Назад ↩️”
- Постоянное меню с эмодзи

## Запуск локально
1. Установите зависимости: `pip install -r requirements.txt`
2. Вставьте токен Telegram в bot.py
3. Запустите: `python bot.py`
